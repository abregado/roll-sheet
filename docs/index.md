# Welcome to Roll Sheet

Roll Sheet is a web application for tracking TTRPG character attributes and rolling dice online with real-time synchronization across all connected players.

## Features

- **Multiple Character Sheets** - Create and manage sheets for all your characters
- **Custom Attributes** - String, integer, and derived attribute types
- **Roll Templates** - Save complex rolls with multiple variants (advantage, disadvantage, etc.)
- **Real-time Sync** - All players see updates instantly
- **Persistent History** - Roll history saved on the server
- **Dice Notation** - Full support including keep/drop modifiers
